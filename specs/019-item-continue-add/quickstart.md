# Quickstart: 商品入力フォームの続けて追加

## Goal
商品追加フォームで 1 件ずつ登録するだけでなく、「続けて追加」で連続入力できることを確認する。

## Prerequisites
- Flutter SDK が利用可能であること。
- `weekly_buyer` プロジェクトが開けること。

## Validate

```powershell
cd weekly_buyer
flutter analyze
flutter test
```

## Manual flow
1. 商品追加画面を開く。
2. 商品名と数量を入力する。
3. 「続けて追加」を押して、フォームが閉じずに入力欄が初期化されることを確認する。
4. 続けて別の商品を入力して、もう一度「続けて追加」を押す。
5. 通常の登録を押して、フォームが閉じることを確認する。
6. ボタンが商品名と数量の入力欄の近くに表示されることを確認する。

## Expected results
- 連続で複数商品を登録できる。
- 「続けて追加」のあともフォームは表示されたままになる。
- 通常の登録ではフォームが閉じる。
- 入力欄とボタンが近くに配置され、操作しやすい。
