# Quickstart: 商品追加画面メモ

## Goal
商品追加画面で、その日の私用メモを入力・編集・保存できることを確認する。

## Prerequisites
- Flutter SDK が利用可能であること。
- `weekly_buyer` プロジェクトを開けること。

## Validate

```powershell
cd weekly_buyer
flutter analyze
flutter test
```

## Manual flow
1. 商品追加画面を開く。
2. その日の私用メモを入力して保存する。
3. 別の日に切り替えて、元の日に戻る。
4. 保存したメモが再表示されることを確認する。
5. メモを編集して保存し直す。
6. メモを空にして保存し、未設定に戻ることを確認する。
7. 購入リスト画面へ移動し、私用メモが表示されないことを確認する。

## Expected results
- 同じ日には保存済みメモが表示される。
- 別の日と混在せず、日ごとに独立して扱われる。
- 空欄保存は未設定として扱われる。
- 購入リスト画面には私用メモが表示されない。
