# Quickstart: 翌週の購入日を既定表示にする

## Goal

商品追加画面を開いたとき、翌週の週表示が初期状態で出ていることを確認する。

## Suggested Checks

1. 依存関係が整っていることを確認する。
2. `flutter analyze` を実行する。
3. `flutter test` を実行する。
4. アプリを起動して商品追加画面を開く。
5. 画面上部の週表示が現在週ではなく翌週になっていることを確認する。
6. 曜日チップのラベルが曜日のみで、選択状態が正しいことを確認する。
7. 左右スワイプで隣の曜日へ移動できることを確認する。
8. 週末や月末にまたがるケースでも翌週表示が崩れないことを確認する。

## Expected Result

商品追加画面は開いた瞬間から翌週の購入を前提にした週を表示し、曜日選択とデータ読み込みがその週に一致する。
