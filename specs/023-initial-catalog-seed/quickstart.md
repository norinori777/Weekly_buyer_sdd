# Quickstart: 初期カテゴリとひらがな名の投入

## Goal

アプリ起動時に指定されたカテゴリと商品候補が投入され、各商品候補にひらがな名が保存されることを確認する。

## Validation Steps

1. データベースを空にした状態でアプリを起動する。
2. 初期カテゴリが指定順で作成されていることを確認する。
3. 各カテゴリに指定された商品候補が作成されていることを確認する。
4. 各商品候補にひらがな名が入っていることを確認する。
5. 既存データがある状態でもう一度アプリを起動し、同じカテゴリや商品候補が重複して増えないことを確認する。

## Test Commands

```bash
cd weekly_buyer
flutter analyze
flutter test test/repository_test.dart
```

## Expected Result

- 指定された初期カテゴリが利用可能になる。
- 各商品候補にひらがな名が登録される。
- 再起動しても seed データが重複しない。
