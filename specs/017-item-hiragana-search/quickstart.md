# Quickstart: 商品名のひらがな候補表示

## Goal
設定画面で商品にひらがなを登録し、商品登録画面で商品名とひらがなの両方から候補を探せることを確認する。

## Prerequisites
- Flutter SDK が利用可能であること。
- `weekly_buyer` プロジェクトが開けること。

## Validate

```powershell
cd weekly_buyer
flutter analyze
flutter test
```

## Manual flow
1. 設定画面を開く。
2. 商品の追加または編集で、商品名とひらがなを入力して保存する。
3. 商品登録画面を開く。
4. 商品名の一部を入力して候補が出ることを確認する。
5. ひらがなの一部を入力して同じ商品候補が出ることを確認する。
6. 商品名とひらがなの両方に一致する候補が 1 件だけ表示されることを確認する。
7. 既存のひらがな未設定商品が商品名検索で引き続き出ることを確認する。

## Expected results
- 設定画面の商品編集にひらがな入力欄が表示される。
- ひらがなが未入力の場合は保存できない。
- 商品登録画面では商品名検索とひらがな検索の両方で候補が出る。
- 同一商品は候補一覧に重複表示されない。
