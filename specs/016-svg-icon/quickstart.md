# Quickstart: SVG アイコン統一

## 目的

アプリのブランド icon が `assets/weekly_buyer.svg` で表示され、主要画面で見た目が崩れないことを確認する。

## 手動確認手順

1. アプリを起動する。
2. 購入リスト、商品追加、設定などの主要画面を開く。
3. AppBar 周辺に SVG のブランド icon が表示されることを確認する。
4. 端末やウィンドウサイズを変えて、icon の比率や見た目が崩れないことを確認する。
5. 主要画面の遷移や操作が変わっていないことを確認する。

## 期待結果

- ブランド icon が PNG ではなく SVG の見た目で表示される。
- 画面サイズが変わっても icon が破綻しない。
- 既存の画面遷移や操作はそのまま使える。
