# Quickstart: 商品登録画面の料理メニュー入力

## Goal
商品登録画面で、朝・昼・夜ごとに料理メニューを追加ボタンから登録し、キャンセルで閉じられることを確認する。

## Prerequisites
- Flutter SDK が利用可能であること。
- `weekly_buyer` プロジェクトが開けること。

## Validate

```powershell
cd weekly_buyer
flutter analyze
flutter test
```

## Manual flow
1. 商品登録画面を開く。
2. 朝・昼・夜の各セクション右上に追加ボタンがあることを確認する。
3. 朝・昼・夜のいずれかの追加ボタンを押して、下からスライドアップする入力フォームが表示されることを確認する。
4. メニューを入力して登録し、選択中のセクションに保存されることを確認する。
5. 追加フォームを開いた状態でキャンセルし、保存されないことを確認する。
6. 別の日に切り替えて、元の日へ戻り、メニューが維持されることを確認する。
7. 購入リスト画面へ移動し、料理メニューが表示されないことを確認する。

## Expected results
- 朝・昼・夜の見出しの下にメニューが表示される。
- 各セクション右上の追加ボタンから入力フォームを開ける。
- キャンセルでフォームを閉じても保存されない。
- 同じ区分に複数メニューを保持できる。
- 購入リスト画面には料理メニューが表示されない。
