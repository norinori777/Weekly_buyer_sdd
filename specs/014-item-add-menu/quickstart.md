# Quickstart: 商品登録画面の料理メニュー入力

## Goal
商品登録画面で、朝・昼・夜ごとに料理メニューを入力、候補選択、削除できることを確認する。

## Prerequisites
- Flutter SDK が利用可能であること。
- `weekly_buyer` プロジェクトが開けること。

## Validate

```powershell
cd weekly_buyer
flutter analyze
flutter test
```

## Manual flow
1. 商品登録画面を開く。
2. 朝・昼・夜の各見出しの下に入力欄があることを確認する。
3. 料理メニューを自由入力し、候補が入力欄の下に表示されることを確認する。
4. 候補を選んでメニューとして追加する。
5. 追加済みメニューの左横の✖を押して個別に消す。
6. 別の日に切り替えて、元の日へ戻り、メニューが維持されることを確認する。
7. 購入リスト画面へ移動し、料理メニューが表示されないことを確認する。

## Expected results
- 朝・昼・夜の見出しの下にメニューが表示される。
- 同じ区分に複数メニューを保持できる。
- 候補を選んで素早く登録できる。
- ✖ ボタンで個別に削除できる。
- 購入リスト画面には料理メニューが表示されない。
